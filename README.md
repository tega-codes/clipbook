# Project: ClipBook - A Community Learning Build**

**Goal:** Build a simple, visual digital scrapbook that lets users save text snippets from any website. 

---

## **üìã Project Layout & Roadmap**

### **Phase 1: Setup & Foundation (Week 1)**
*Everyone does this on their local machine*

**Search/Research Topics:**
- "How to set up a basic GitHub repository"
- "Basic HTML structure with header, main, footer"
- "CSS Flexbox basics for layout"
- "JavaScript arrays and objects basics"

**Tasks:**
1. **Repository Setup** (one person)
   - Create GitHub repo named `clipbook-project`
   - Add `README.md` with project description
   - Create basic folder structure: `/public` for frontend, `/docs` for notes

2. **Local Development Setup** (everyone)
   - Fork the repo to your own GitHub
   - Clone it locally
   - Create `index.html`, `style.css`, `script.js` in `/public`
   - Set up a basic HTML5 template

3. **First Contribution** (everyone)
   - Add your name to a `CONTRIBUTORS.md` file via pull request
   - Learn: Git clone, commit, push, pull request workflow
